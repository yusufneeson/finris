# FINRIS

**Finance Record and Insight System**

FINRIS adalah aplikasi pencatatan keuangan pribadi.
Saya membangun ini dari 0 dengan tech stack berikut:
  - TypeScript
  - Next.js (Main framework)
  - TailwindCSS (Css framework)
  - Arcticjs (Oauth2)
  - Drizzle (Database ORM)
